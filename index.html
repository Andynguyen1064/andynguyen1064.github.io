About

I’m an aspiring data analyst/product analyst focused on building practical analytics tools, dashboards, and SQL-based insights. My projects combine business questions, data cleaning, KPI design, visualization, and technical implementation.

How I Approach Analytics Projects

My projects follow a practical analytics workflow from raw data to stakeholder-ready insights.

01

Define the Question

Start with the business problem, user need, or decision the dashboard should support.

02

Prepare the Data

Clean, join, validate, and structure the data so metrics are reliable and repeatable.

03

Build Metrics

Define KPIs, filters, cohorts, and logic clearly so stakeholders know what each number means.

04

Visualize Insights

Create dashboards, charts, and tables that make trends, risks, and opportunities easier to understand.
